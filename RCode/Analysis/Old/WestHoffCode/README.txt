The folder `Data and R Code' contains two data sets and two R code files.  These data and code were used in the paper ``A Mixed Effects Model for Longitudinal Relational and Network Data, with Applications to International Trade and Conflict'':

1.) Data:
a.) Trade.csv
b.) MID.csv - This data set was updated on 10/9/2011 to coincide with the published paper.

2.) R Code:
a.) LSRTradeGRW.R
b.) LSNProbitMID.R
